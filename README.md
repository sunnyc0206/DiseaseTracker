# World Disease Tracker (WDT)

World Disease Tracker (WDT) is a modern web application for tracking global disease outbreaks and health emergencies across countries and regions in real time. It aggregates up-to-date epidemiological data, visual maps, and trend graphs to help users monitor risks and patterns worldwide. WDT also integrates the latest news and alerts from authoritative sources such as the CDC, WHO, and other public health agencies to keep you informed of current developments.
